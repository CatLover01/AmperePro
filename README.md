# AmpèrePro

AmpèrePro is a learning application focused on electricity concepts, based on real-time simulation of electrical circuits. It allows you to explore fundamental concepts through a free mode and a game mode, making learning more interactive.

The project aims to help students and teachers better understand electrical circuits, their components, and their behaviors in different scenarios.

## Prerequisites

- Python (3.12+)
- PySide6
- NumPy
- dacite

## Installation

Install the necessary packages with:

```bash
pip install -r requirements.txt
```

The entry point for the application is `main.py`.

## Tests

Unit tests are located in the `tests/` folder and can be run with:

```bash
python -m unittest discover tests
```
